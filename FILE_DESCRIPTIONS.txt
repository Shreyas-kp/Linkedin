src/services/
- Client-side services (mostly TypeScript) that implement business logic, integrations, and adapters. Key files:

  localGenerator.ts
  - A local in-browser generator: likely contains heuristics or a lightweight model to produce post drafts without external calls.

  PostGenerationService.ts
  - Higher-level wrapper around generation flows; may call localGenerator, RAGService, or external endpoints if configured.

  RAGService.ts
  - Retrieval-Augmented Generation helper: fetches style examples from the style library or vector store and uses them to guide generation.

  vectorStore.ts
  - Simple vector store adapter for storing and retrieving embeddings; may wrap `idb` or in-memory store.

  hashtagSuggester.ts
  - Service that suggests hashtags for a given post text (heuristics or small ML model).

  engagementPredictor.ts
  - Predicts engagement score for a post. There is an ML model under `ml/` that likely relates to this service.

  toxicityChecker.ts
  - Runs a lightweight toxicity check on post content and flags unsafe text.

src/utils/

  postFormatting.ts
  - Utilities for formatting a post (escaping, removing excess whitespace, preparing CSV lines, etc.). Used by export or copy functions.

  enhancedGenerator.ts
  - Helper utilities used by EnhancedPostGenerator. Might contain text transformation helpers.

src/data/

  examples.json
  - JSON file with example posts or writing-style examples used by the style library and RAG retrieval.
